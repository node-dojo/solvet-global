# Assets

This directory contains static assets for the SOLVET Global project.

## Structure

### `renders/`
Rendered images and visual assets.

### `previews/`
Preview images and thumbnails.

## Usage

Assets in this directory are used for:
- Documentation
- Testing
- Development
- Reference

## Adding Assets

1. Place assets in appropriate subdirectory
2. Use descriptive filenames
3. Optimize file sizes when possible
4. Document asset purpose if needed

## Git Considerations

Large binary files should be:
- Optimized before committing
- Considered for Git LFS if very large
- Documented in `.gitignore` if temporary
