# Development Tools

This directory contains development tools and utilities for the SOLVET Global project.

## Tools

### `threejs-dev/`
Three.js development environment for testing and prototyping 3D features.

**Usage:**
```bash
cd tools/threejs-dev
npm install
npm run dev
```

### `solvet-cli/`
SOLVET Command Line Interface (deprecated).

**Note**: This CLI has been replaced with Cursor/Claude natural language commands. See [CURSOR_COMMANDS.md](../docs/guides/cursor-commands.md) for the new command system.

The CLI code is preserved for reference but is no longer actively maintained.

## Adding Tools

1. Create tool directory in `tools/`
2. Add README explaining purpose and usage
3. Update this index
4. Follow project standards
