# NO3D Tools Website

E-commerce website for NO3D Tools subscription system.

## Pages

- `index.html` - Main landing page (existing)
- `subscribe.html` - Subscription landing page
- `success.html` - Post-checkout success page
- `account.html` - Account management page

## Features

- Subscription checkout integration with Polar
- Account management with subscription status
- Library download functionality
- Update notifications

## Setup

1. Ensure API endpoints are deployed to Vercel
2. Update API URLs in JavaScript if needed
3. Configure subscription checkout link in `subscribe.html`
4. Deploy to Vercel

## Configuration

The website reads subscription configuration from `/config/subscription-config.json` which is generated by the setup script.

## API Integration

The website integrates with these API endpoints:
- `/api/v1/user/entitlements` - Get subscription status
- `/api/v1/library/download` - Download library ZIP
- `/api/v1/catalog/version` - Get catalog version
