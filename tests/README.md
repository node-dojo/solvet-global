# Tests

This directory contains test files and utilities for the SOLVET Global project.

## Test Files

### `check-glb-error.py`
Utility script for checking GLB file errors.

### `test-embed-generation.py`
Test script for 3D embed generation.

### `preview-polar-checkout.html`
Preview/test file for Polar checkout integration.

## Running Tests

### Python Tests
```bash
python tests/check-glb-error.py
python tests/test-embed-generation.py
```

### HTML Preview
Open `tests/preview-polar-checkout.html` in a browser.

## Test Guidelines

- Keep test files focused and isolated
- Document test purpose and usage
- Clean up test artifacts
- Don't commit sensitive test data

## Adding Tests

1. Create test file in this directory
2. Document purpose and usage
3. Add to this README
4. Follow project testing standards
